# ULTIMOROBO

## Institutional Portfolio Governance Platform

ULTIMOROBO é uma plataforma institucional de governança de portfólio.

O sistema não prevê mercado.

O sistema não gera sinais especulativos.

O sistema avalia:

- Regime macroeconômico
- Alocação estratégica
- Orçamento de risco
- Liquidez
- Contrapartes
- Stress sistêmico
- Sobrevivência operacional
- Governança
- Auditoria determinística
- Auditoria histórica
