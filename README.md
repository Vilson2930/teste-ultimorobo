# ULTIMOROBO

## Sistema Institucional de Alocação Macro Global

---

# Visão Executiva

O ULTIMOROBO é um sistema institucional de gestão de portfólio baseado em regimes macroeconômicos.

Seu objetivo não é prever preços, identificar topos ou fundos, nem gerar sinais especulativos.

O sistema busca:

- Preservação de capital
- Controle de risco
- Sobrevivência de longo prazo
- Alocação baseada em evidências
- Governança quantitativa
- Decisões auditáveis

Toda decisão deve ser explicável, reproduzível e validável.

---

# Missão

Transformar informações macroeconômicas, riscos de mercado e métricas de governança em decisões objetivas de alocação.

O sistema foi construído para responder uma única pergunta:

> Como distribuir capital entre ativos de forma racional e auditável?

---

# Filosofia do Projeto

O ULTIMOROBO segue cinco princípios fundamentais.

### 1. Sobrevivência primeiro

Antes de buscar retorno, o sistema protege o patrimônio.

---

### 2. Macro define direção

Liquidez global e política monetária são os principais direcionadores dos mercados.

---

### 3. Risco controla exposição

Nenhuma posição é analisada isoladamente.

O sistema avalia:

- risco individual
- risco agregado
- risco sistêmico

---

### 4. Governança acima de opinião

Nenhuma decisão é baseada em narrativas.

Toda decisão deve passar pelos motores de governança.

---

### 5. Processo acima de previsão

O sistema não tenta prever mercado.

O foco é:

- adaptação
- disciplina
- consistência
